Article Management 2.0 RC for XOOPS 2.3+

Publishing Platform and Content Management System

— Article Dev Group

System Recommendations

  • PHP version 4.1 or higher
  • MySQL version 3.23 or higher
  • Xoops 2.3.0 or higher
  • Frameworks released with XOOPS 2.30+ core package

Installation

  1. Unzip the package in an empty directory
  2. Upload the folder "article" to Your-Xoops-Root-Path/modules/
  3. Install "article" from your XOOPS => admin => Modules.
  4. Set permissions, blocks and configs as a usual module
