Help: XMF Demo

Description

The XMF stands for XOOPS Module Framework, and its goal is to make development for XOOPS modules easy and fun.

This is a demo showing the basic functionality of different features, like InfoBox and ButtonBox in the About page.

Install/uninstall

No special measures necessary, follow the standard installation process for modules â€“ extract the /xmf folder into the /modules directory. Install the module through Admin -> System Module -> Modules.

Then install the XMF-based module, and you're ready to go

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Operating instructions

The XMF module itself doesn't have any configuration at the moment. Of course, the individual XMF-based modules might have tons of configuration options, but it will be up to their developers.
