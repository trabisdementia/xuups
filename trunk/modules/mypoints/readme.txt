What is mypoints Module ?
======================

With this Xoops module, your can give you members points for each submission or vote.

The module can :
. display an ordered list of your top members
. etc


How to install mypoints
====================

Copy mypoints folder into the /modules directory of your website. Then log in
to your site as administrator, go to System Admin > Modules, look for the mypoints
icon in the list of uninstalled modules and click in the install icon. Follow
the directions in the screen and you'll be ready to go.

This module makes use of plugins. Without plugins there will be nothing to see.
Available plugins are located at mypoints/plugins.

At this moment we provide plugins for:
arms, forum, news, mydownloads, mylinks, system(for coments), tutorials, webshow, wfsections and xoopstube.


Requirements
====================

It should work with xoops 2.2.x, xoops 2.0.x, xoops 2.3.0, impresscms and Xcl.

It should work with php4 and php5.

Tested only in impresscms 1.0 and xoops 2.3.0 under a php5 enviroment.


Developing plugins
====================

With little effort you can adapt existing plugins to work with a your module.
If you are a module developer you can include your plugins under yourmodule/include/mypoints.plugin.php
The plugin should also work for modules that use XOOPS_TRUST_PATH.

You can send your plugin to lusopoemas@gmail.com if you want me to add it to mypoints/plugins folder.
